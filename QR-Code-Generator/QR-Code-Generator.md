# QR Code Generator (Python)

A simple Python project that generates a QR code from any URL and saves it as a PNG image.

---

## 📌 Features

- Generate QR codes from URLs   
- Save QR code as a PNG image
- Easy to use
- Beginner-friendly project

---

## 📂 Project Structure

```text
QR-Code-Generator/
│
├── main.py
├── qrcode.png      # Generated after running the program
└── README.md
```

---

## 📦 Requirements

- Python 3.8 or higher
- qrcode library
- Pillow library

---

## 📥 Installation

### Install the required package

```bash
py -m pip install qrcode[pil]
```
