# Tic-Tac-Toe Game

A simple **Tic-Tac-Toe** game built using **Python** and **Tkinter**. Two players can play alternately as **X** and **O** with automatic winner detection, draw detection, and a reset option.

---

## 📌 Features

- Two-player game (X and O)
- Automatic winner detection
- Draw detection
- Reset button to start a new game
- Beginner-friendly GUI using Tkinter

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns clicking an empty square.
3. The first player to get **three symbols in a row** wins.
4. If all squares are filled without a winner, the game ends in a draw.
5. Click **Reset** to start a new game.
